# Sando Design System

Welcome to the Sando Design System documentation!

## ðŸŽ¨ Design Philosophy

Sando follows a three-layer token architecture inspired by the Japanese "katsu sando" sandwich:

### 1. **Ingredients** (Primitives)
Raw, atomic values with no opinion:
- Colors: `color-brand-500`, `color-neutral-100`
- Spacing: `space-base`, `space-2x`
- Typography: `font-size-base`

### 2. **Flavors** (Semantic)
Context and meaning to ingredients:
- `color-background-interactive`
- `color-text-primary`
- `spacing-component-gap`

### 3. **Recipes** (Component-specific)
Tokens consumed by components:
- `button-background-color`
- `button-padding-horizontal`
- `card-border-radius`
