The Sando Design System typography foundation provides a comprehensive type scale, font families, weights, and spacing controls built on **Ingredients tokens** (Layer 1).

## Usage Guidelines

### Do's

- ✅ Use the type scale consistently across all components
- ✅ Use sans-serif for UI text, monospace for code
- ✅ Apply appropriate line heights for content type
- ✅ Use font weights to create hierarchy

### Don'ts

- ❌ Don't use arbitrary font sizes outside the scale
- ❌ Don't mix too many weights in a single interface
- ❌ Don't use tight line heights for body text
- ❌ Don't use excessive letter spacing on body text

---

## Accessibility

- Minimum font size for body text: `font-size-200` (14px)
- Line height for body text should be at least 1.5 (`lineHeight-150`)
- Ensure sufficient color contrast (see Color foundation)
- Use relative units (rem) for responsive scaling
