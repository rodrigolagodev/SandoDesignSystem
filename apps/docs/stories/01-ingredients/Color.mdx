# Color

The Sando Design System provides **8 refined color palettes** + **3 neutral variants**, all carefully crafted using **OKLCH color space** for perceptual uniformity. Inspired by the Japanese Sando aesthetic—minimalist, sophisticated, and artisanal—our colors are intentionally **less saturated** than typical design systems, creating a calm, premium feel.

## Color Philosophy: The Sando Aesthetic

Sando's color system is inspired by the Japanese sandwich (サンド):

- **Soft, Not Loud**: Lower chroma values (~40-50% less than typical) for sophistication
- **Warm & Natural**: Hues shifted toward earthy, organic tones
- **Refined Minimalism**: Colors that whisper, not shout
- **WCAG AA Guaranteed**: All palettes pass accessibility standards
- **11 Shades**: From 50 (near-white) to 950 (near-black)

## Usage Guidelines

### Do's

- Use Ingredients tokens directly in rare cases when building new Flavor tokens
- Reference brand colors for primary actions and brand moments
- Use neutral colors for text, backgrounds, and UI structure
- Apply state colors consistently across all feedback scenarios

### Don'ts

- Don't use Ingredients tokens directly in components (use Recipes instead)
- Don't mix color systems (stay within the design system)
- Don't use hardcoded color values in your code
- Don't use state colors for decorative purposes

## Technical Details: OKLCH Values

Our refined palette uses intentionally lower chroma values:

<table>
  <thead>
    <tr>
      <th>Profile</th>
      <th>Typical DS</th>
      <th>Sando DS</th>
      <th>Reduction</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Peak Chroma</td>
      <td>0.17-0.22</td>
      <td>0.10-0.14</td>
      <td>~40%</td>
    </tr>
    <tr>
      <td>Light tints (50-100)</td>
      <td>0.06-0.09</td>
      <td>0.015-0.03</td>
      <td>~75%</td>
    </tr>
  </tbody>
</table>

**Why lower saturation?**

- Creates visual calm and reduces eye strain
- Feels more premium and sophisticated
- Better harmony between colors
- Still maintains WCAG AA contrast

---

## Accessibility

All color combinations in our Flavors layer meet **WCAG 2.1 AA** standards for color contrast:

- **Text**: Minimum 4.5:1 contrast ratio
- **Large text**: Minimum 3:1 contrast ratio
- **UI components**: Minimum 3:1 contrast ratio

The color system is designed to work seamlessly in both light and dark modes.
