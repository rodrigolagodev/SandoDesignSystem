# Button Tokens

Component-specific tokens for `<sando-button>`. These Recipe tokens ensure buttons look and behave consistently across all flavors and themes.

## Token Reference

### Solid Variant

The primary, high-emphasis button style.

| Token | Purpose | Default Value |
|-------|---------|---------------|
| `--sando-button-solid-backgroundColor-default` | Default background | `var(--sando-color-action-solid-background-default)` |
| `--sando-button-solid-backgroundColor-hover` | Hovered background | `var(--sando-color-action-solid-background-hover)` |
| `--sando-button-solid-backgroundColor-active` | Pressed background | `var(--sando-color-action-solid-background-hover)` |
| `--sando-button-solid-backgroundColor-disabled` | Disabled background | `var(--sando-color-action-disabled-background)` |
| `--sando-button-solid-textColor-default` | Default text color | `var(--sando-color-action-solid-text-default)` |
| `--sando-button-solid-textColor-disabled` | Disabled text color | `var(--sando-color-action-disabled-text)` |

### Outline Variant

The secondary, medium-emphasis button style with borders.

| Token | Purpose | Default Value |
|-------|---------|---------------|
| `--sando-button-outline-backgroundColor-hover` | Hovered background | `var(--sando-color-background-emphasis)` |
| `--sando-button-outline-backgroundColor-active` | Pressed background | `var(--sando-color-background-emphasis)` |
| `--sando-button-outline-backgroundColor-disabled` | Disabled background | `var(--sando-color-action-disabled-background)` |
| `--sando-button-outline-textColor-default` | Default text color | `var(--sando-color-action-outline-text-default)` |
| `--sando-button-outline-textColor-disabled` | Disabled text color | `var(--sando-color-action-disabled-text)` |
| `--sando-button-outline-borderColor-default` | Default border | `var(--sando-color-action-outline-border-default)` |
| `--sando-button-outline-borderColor-hover` | Hovered border | `var(--sando-color-action-outline-border-hover)` |
| `--sando-button-outline-borderColor-disabled` | Disabled border | `var(--sando-color-action-disabled-border)` |

### Ghost Variant

The tertiary, low-emphasis button style without borders.

| Token | Purpose | Default Value |
|-------|---------|---------------|
| `--sando-button-ghost-backgroundColor-hover` | Hovered background | `var(--sando-color-action-ghost-background-hover)` |
| `--sando-button-ghost-backgroundColor-active` | Pressed background | `var(--sando-color-action-ghost-background-hover)` |
| `--sando-button-ghost-backgroundColor-disabled` | Disabled background | `transparent` |
| `--sando-button-ghost-textColor-default` | Default text color | `var(--sando-color-action-ghost-text-default)` |
| `--sando-button-ghost-textColor-disabled` | Disabled text color | `var(--sando-color-action-disabled-text)` |

## Customization Examples

### Example 1: Danger Button

```css
.button--danger {
  --sando-button-solid-backgroundColor-default: var(--sando-color-state-error-500);
  --sando-button-solid-backgroundColor-hover: var(--sando-color-state-error-600);
  --sando-button-solid-textColor-default: var(--sando-color-utility-white);
}
```

```html
<sando-button class="button--danger">Delete</sando-button>
```

### Example 2: Success Button

```css
.button--success {
  --sando-button-solid-backgroundColor-default: var(--sando-color-state-success-500);
  --sando-button-solid-backgroundColor-hover: var(--sando-color-state-success-600);
  --sando-button-solid-textColor-default: var(--sando-color-utility-white);
}
```

```html
<sando-button class="button--success">Confirm</sando-button>
```

### Example 3: Per-Flavor Customization

```css
/* Ocean flavor uses different button colors */
[flavor="ocean"] {
  --sando-button-solid-backgroundColor-default: var(--sando-color-blue-600);
  --sando-button-solid-backgroundColor-hover: var(--sando-color-blue-700);
}
```

```html
<div flavor="ocean">
  <sando-button variant="solid">Ocean Button</sando-button>
</div>
```

### Example 4: Custom Outline Style

```css
.button--custom-outline {
  --sando-button-outline-borderColor-default: var(--sando-color-purple-500);
  --sando-button-outline-borderColor-hover: var(--sando-color-purple-700);
  --sando-button-outline-textColor-default: var(--sando-color-purple-700);
}
```

## Accessibility Notes

All button tokens maintain WCAG 2.1 AA contrast requirements:
- **Solid variant**: 4.5:1 text contrast on background
- **Outline variant**: 3:1 border contrast, 4.5:1 text contrast
- **Ghost variant**: 4.5:1 text contrast on page background
- **Focus ring**: Automatic via `--sando-color-focus-ring`

The button component automatically includes:
- Focus visible indicators
- Disabled state with reduced opacity
- Keyboard support (Enter, Space)
- Screen reader accessible labels
