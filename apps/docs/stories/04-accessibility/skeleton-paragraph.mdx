# Accessibility — sando-skeleton-paragraph

> Source: [`packages/components/src/components/skeleton-paragraph/ACCESSIBILITY.md`](https://github.com/rodrigolagodev/SandoDesignSystem/blob/master/packages/components/src/components/skeleton-paragraph/ACCESSIBILITY.md)

## ARIA pattern

Follows the presentation/none pattern — the skeleton paragraph is a loading placeholder hidden from AT.

  <thead>

`sando-skeleton-paragraph` composes multiple `sando-skeleton` instances.

## Screen reader behavior

All skeleton lines are invisible to AT. Recommended pattern:

```html
<div aria-busy="true" aria-label="Loading article">
  <sando-skeleton-paragraph lines="4"></sando-skeleton-paragraph>
</div>
```

## Color contrast

Decorative — hidden from AT. Contrast not applicable for accessibility purposes.
