# Accessibility — sando-select

> Source: [`packages/components/src/components/select/ACCESSIBILITY.md`](https://github.com/rodrigolagodev/SandoDesignSystem/blob/master/packages/components/src/components/select/ACCESSIBILITY.md)

## ARIA pattern

Follows the [WAI-ARIA Combobox](https://www.w3.org/WAI/ARIA/apg/patterns/combobox/) pattern with a listbox popup.

## Screen reader behavior

Closed: "Country, combobox, collapsed". Open: "Country, combobox, expanded". Navigating: "France, 5 of 10". Selected: "France, selected".

## Color contrast

Verified via `jest-axe` against the `original` flavor in light and dark modes. Disabled state uses intentional reduced contrast.

## Tested with

- jest-axe (automated, runs in `sando-select.a11y.test.ts`)
- VoiceOver (macOS Sequoia, Safari 18)
- NVDA (Windows 11, Firefox 130)

## Known limitations

None identified.
