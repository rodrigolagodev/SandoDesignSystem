# Accessibility — sando-option

> Source: [`packages/components/src/components/option/ACCESSIBILITY.md`](https://github.com/rodrigolagodev/SandoDesignSystem/blob/master/packages/components/src/components/option/ACCESSIBILITY.md)

## ARIA pattern

Follows the [WAI-ARIA Listbox](https://www.w3.org/WAI/ARIA/apg/patterns/listbox/) pattern — the `option` role within a `listbox`.

<table>
  <thead>
    <tr>
      <th>Attribute</th>
      <th>Value</th>
      <th>Purpose</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>`role`</td>
      <td>`option`</td>
      <td>Marks the element as a selectable option in a listbox</td>
    </tr>
    <tr>
      <td>`aria-selected`</td>
      <td>`"true"` / `"false"`</td>
      <td>Communicates the selected state</td>
    </tr>
    <tr>
      <td>`aria-disabled`</td>
      <td>`"true"`</td>
      <td>Marks the option as non-interactive when disabled</td>
    </tr>
  </tbody>
</table>

Designed to be used inside `sando-select`. Not intended for standalone use outside a listbox context.

## Keyboard map

<table>
  <thead>
    <tr>
      <th>Key</th>
      <th>Behavior</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>ArrowUp / ArrowDown</td>
      <td>Navigate between options (managed by parent `sando-select`)</td>
    </tr>
    <tr>
      <td>Enter / Space</td>
      <td>Select the highlighted option (managed by parent)</td>
    </tr>
    <tr>
      <td>Home</td>
      <td>Move highlight to first option (managed by parent)</td>
    </tr>
    <tr>
      <td>End</td>
      <td>Move highlight to last option (managed by parent)</td>
    </tr>
  </tbody>
</table>

Keyboard navigation uses `aria-activedescendant` managed by the parent `sando-select`.

## Screen reader behavior

When highlighted: "Option label, 3 of 10". Selected: "Option label, selected". Disabled: "Option label, dimmed".

## Color contrast

Verified via `jest-axe` against the `original` flavor in light and dark modes. Disabled state uses intentional reduced contrast.

## Tested with

- jest-axe (automated, runs in `sando-option.a11y.test.ts`)
- VoiceOver (macOS Sequoia, Safari 18)
- NVDA (Windows 11, Firefox 130)

## Known limitations

None identified.
