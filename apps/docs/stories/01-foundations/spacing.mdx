# Spacing

The Sando Design System uses a consistent spacing scale based on a **4px base unit** (0.25rem). This creates visual rhythm and maintains consistency across all components and layouts.

## Spacing Scale

Our spacing system uses numeric tokens that follow a logical progression, making it easy to create balanced layouts.

## Spacing Principles

### The 4px Base Unit

All spacing values are multiples of 4px (0.25rem), creating a consistent rhythm:

- **Small spaces (1-4)**: Fine-tune dense UI elements
- **Medium spaces (5-12)**: Standard component spacing
- **Large spaces (13-24)**: Section and layout spacing
- **XL spaces (32-64)**: Page-level layout and whitespace

### Consistent Rhythm

Using a predictable scale helps create visual balance and makes design decisions easier.

<div style={{ marginTop: '24px', padding: '16px', backgroundColor: 'var(--sando-color-background-emphasis)', borderRadius: '8px' }}>
  <strong>Example hierarchy:</strong>
  <ul style={{ marginTop: '8px', marginBottom: '0', fontSize: '14px', lineHeight: '1.6' }}>
    <li>Tight: <code>space-1</code> (4px) - Related items</li>
    <li>Default: <code>space-4</code> (16px) - Component internal spacing</li>
    <li>Comfortable: <code>space-6</code> (24px) - Between components</li>
    <li>Spacious: <code>space-8</code> (32px) - Between sections</li>
  </ul>
</div>

## Usage Guidelines

### Do's

- ✅ Use tokens from the spacing scale for all margins, padding, and gaps
- ✅ Prefer larger spacing values for better readability
- ✅ Use consistent spacing within similar components
- ✅ Create visual hierarchy with spacing variation

### Don'ts

- ❌ Don't use arbitrary spacing values outside the scale
- ❌ Don't use too-tight spacing that reduces readability
- ❌ Don't mix spacing systems (stick to the 4px base)
- ❌ Don't use spacing tokens for sizing (use sizing tokens instead)

---

## Responsive Spacing

While these tokens provide fixed values, you can use CSS clamp() or media queries to adjust spacing responsively:

```css
.element {
  /* Responsive padding */
  padding: clamp(var(--sando-space-4), 3vw, var(--sando-space-8));
}
```
