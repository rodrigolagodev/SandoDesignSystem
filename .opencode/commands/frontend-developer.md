# Frontend Developer Command

Quick alias to invoke the `sando-developer` agent for component work.

## Usage

```
/frontend-developer <task>
```

## What This Does

Delegates to the `sando-developer` agent which handles:

- Component implementation (Lit 3+ / TypeScript 5+)
- 7-file pattern scaffolding
- Keyboard navigation & ARIA
- Recipe token styling

## Examples

```
/frontend-developer Implement the Checkbox component
/frontend-developer Add loading state to Button
/frontend-developer Fix focus ring on Input component
```
